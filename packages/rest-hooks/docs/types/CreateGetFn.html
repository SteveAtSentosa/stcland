CreateGetFn: ((restPath: string, restParams: RestParams, axiosOptions?: Partial<AxiosRequestConfig>) => GetFn)

Creates a function that performs a REST GET request for the specified restPath and restParams. The returned function can be passed directly into 'react-query' query hooks.

Type declaration

    • (restPath, restParams, axiosOptions?): GetFn

    • Parameters

      • restPath: string

        Path to the REST endpoint. Can contain path variables in the form of :variableToSubstitute.

      • restParams: RestParams

        Object containing path and query parameters for the request.

      • OptionalaxiosOptions: Partial<AxiosRequestConfig>

        Optional Axios request configuration.

      Returns GetFn

      A function that can be called to perform a REST GET request.

See

  • RestParams for detailed structure.
  • documentation for available options.
  • GetFn for details on the returned function.

Example

// Basic usage
  const getThings = restClient.createGetFn("/things")
  const things = await getThings()

Example

// With a specific ID in the path
  const getThing1 = restClient.createGetFn("/things/1")
  const thing1 = await getThing1()

Example

// Using path variables
  const getThing1WithPathVariable = restClient.createGetFn("/things/:id", {
    pathParams: { id: 1 }
  })
  const thing1WithVar = await getThing1WithPathVariable()

Remarks

The restParams object can include both pathParams and queryParams. Path variables in restPath (e.g., :id) will be substituted with values from pathParams. Query parameters from queryParams will be appended to the URL.
